div垂直水平居中的N种方法

发表于 2017-11-20

前言

div动态垂直水平居中的方法有很多种，每种实现的方法都值得学习，今天就在这里总结下。

注意布局

(已知div高度和宽度的情况下)

方法一

这种方法对于我来说不常用，先Mark

.content {
    position: relative;
}
.content-div {
    position: absolute;
    margin: auto;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
}

方法二

这种方法很好理解，用得也比较多，运用百分比和宽高的负一半实现水平垂直居中

.content {
    position: relative;
}
.content-div {
    position: absolute;
    top: 50%;
    left :50%;
    margin-left: -(宽度1/2)px;
    margin-top : -(高度1/2)px;
}

不知道div高度和宽度情况下

方法一

display-table 比较经典的居中方法

方法二

css transform，不方便的是根据浏览器不同需要写些前缀

方法三

这是最快最方便的方法了，写flex布局建议配合postcssd的Autoprefixer插件来写，效率会很高

结语

以上都是很实用的div垂直水平居中方法，多练练，多写写，多积累
